This is a task scheduler webapp done using REST. Works with explicit url
localhost:8000/task to view all tasks
localhost:8000/completed_task to view checked out tasks
localhost:8000/due_task to view to do tasks

Added User Auth system
localhost:8000/register
localhost:8000/api-auth

Register using a username  and pass. The click on LOGIN, then login usng the credentials you used for registration
You cannot view the contents without proper login
